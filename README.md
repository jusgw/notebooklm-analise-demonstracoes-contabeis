# notebooklm-analise-demonstracoes-contabeis
Projeto desenvolvido no Bootcamp Afya – Automação de Dados com IA utilizando o NotebookLM como ferramenta de aprendizagem ativa para estudar Balanço Patrimonial, DRE e análise financeira. O projeto reúne curadoria de fontes, engenharia de prompts, resumos estruturados e aplicações práticas voltadas ao mercado financeiro.
